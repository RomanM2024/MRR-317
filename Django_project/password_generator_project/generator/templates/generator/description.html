{% load static %} Генератор паролей

Генератор паролей - это простой веб-сайт, который генерирует случайный пароль на основе выбранных пользователем критериев. Пользователь может выбирать между прописными буквами, специальными символами и цифрами. Кроме того, пользователь может выбрать длину сгенерированного пароля. Приложение будет запускаться в браузере и содержать динамически обновляемые HTML и CSS, основанные на коде Django.
